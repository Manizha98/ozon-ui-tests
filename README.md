Ozon UI Tests

Проект автоматизации тестирования веб-приложения с использованием Python, Pytest и Playwright.

Описание

Репозиторий содержит набор автоматизированных тестов для проверки основных сценариев работы веб-приложения.

Используемые технологии

Python
Pytest
Playwright
GitHub

Реализованные проверки

Проверка открытия главной страницы
Проверка корректности URL
Проверка доступности веб-страницы
Проверка заголовка страницы

Структура проекта

ozon-ui-tests/

├── test_search.py

└── README.md

Установка зависимостей

pip install pytest playwright

Запуск тестов

pytest

Автоматизация поиска товаров

Проверка корзины
Реализация Page Object Model
